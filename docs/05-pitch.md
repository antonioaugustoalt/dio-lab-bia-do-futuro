# 🎤 PC Builder Expert

## 📌 Pitch (3 minutos)

### 1. O Problema (30s)
Montar um computador é uma tarefa complexa e confusa para a maioria das pessoas.  
O usuário precisa:
- Entender compatibilidade entre peças  
- Equilibrar desempenho e orçamento  
- Escolher entre centenas de opções  

Mesmo assim, é comum:
- Gastar mal o dinheiro  
- Montar PCs com gargalo  
- Comprar configurações inadequadas para seu uso  

👉 Existe uma grande dificuldade em transformar uma necessidade simples — *“quero um PC bom”* — em uma decisão técnica correta.

---

### 2. A Solução (1 min)
Criamos o **PC Builder Expert**, um agente de IA que funciona como um especialista em montagem de computadores.

**Diferenciais:**
- Entende a necessidade do usuário a partir de uma pergunta simples  
- Identifica automaticamente o perfil de uso (gamer, programador, IA, etc.)  
- Analisa o orçamento  
- Gera uma configuração completa, peça por peça  

**Baseado em:**
- Regras técnicas reais  
- Compatibilidade entre hardware  
- Base de dados estruturada  

Além disso, utiliza um modelo inteligente de busca (**RAG**) que seleciona apenas os componentes mais relevantes, tornando a recomendação rápida, precisa e escalável.  
👉 O usuário não precisa entender nada de hardware — o agente faz todo o trabalho.

---

### 3. Demonstração (1 min)
Exemplo prático:  
Usuário pergunta: *“Quero um PC gamer até 5 mil reais”*

O agente:
1. Interpreta objetivo e orçamento  
2. Identifica perfil como gamer  
3. Seleciona os melhores componentes disponíveis  
4. Retorna uma configuração completa  

Inclui:
- CPU  
- GPU  
- Memória  
- Armazenamento  
- Fonte  
- Gabinete  

Além disso, explica o motivo de cada escolha, garantindo transparência e confiança.  
👉 Tudo isso em poucos segundos, sem necessidade de conhecimento técnico.

---

### 4. Diferencial e Impacto (30s)
O grande diferencial é combinar:
- Inteligência artificial  
- Regras técnicas reais  
- Dados estruturados  

**Impacto direto:**
- Democratiza o acesso à tecnologia  
- Evita desperdício de dinheiro  
- Melhora a qualidade das compras  
- Aplicável em e-commerces, suporte técnico ou consultoria automatizada  

👉 Transformamos um processo complexo em uma experiência simples, rápida e acessível para qualquer pessoa.
